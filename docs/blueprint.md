# **App Name**: IntraVision

## Core Features:

- Room Scan & Upload: Allow users to scan their room using their phone's camera and upload it to the platform for analysis.
- AI-Driven Furniture Recommendations: Provide furniture recommendations based on room analysis, user style preferences, and budget, acting as a recommendation tool.
- AR Furniture Visualization: Enable users to visualize selected furniture items in their room using augmented reality.
- Mood-Based Suggestions: Generate interior design and furniture suggestions based on user-specified moods, acting as a recommendation tool.
- Virtual Room Layout: Allow users to re-arrange the visualized furniture, and automatically suggest optimized room layouts based on the AI and room size.

## Style Guidelines:

- Primary color: A muted, sophisticated blue (#6A89CC), reflecting trust and innovation in design.
- Background color: A very light, desaturated blue (#F0F4FF), creating a calming and spacious feel.
- Accent color: A contrasting warm orange (#D46A4E) for calls to action and highlighting key features.
- Body and Headline Font: 'Inter', a sans-serif font offering a clean and modern aesthetic suitable for both headlines and body text.
- Use minimalist, line-based icons for a modern, uncluttered look. Icons should be intuitive and easily recognizable.
- A clean and spacious layout with a focus on visual clarity. Prioritize ease of navigation and intuitive user flows.
- Subtle transitions and animations to enhance user experience, such as fading effects and smooth scrolling.
